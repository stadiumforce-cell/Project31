1. ΕΙΣΑΓΩΓΗ
Το παρόν project υλοποιεί ένα πλήρες Σύστημα Διαχείρισης Δανειστικής
Βιβλιοθήκης σε Python, με γραφικό περιβάλλον χρήστη (GUI) μέσω της
βιβλιοθήκης tkinter. Το σύστημα υποστηρίζει:

• Διαχείριση μελών  
• Διαχείριση βιβλίων  
• Διαχείριση δανεισμών  
• Στατιστικά χρήσης  
• Σύστημα προτάσεων βιβλίων (Recommendation System)  
• Αποθήκευση και φόρτωση δεδομένων από αρχεία .txt  

Το project είναι πλήρως modular, με διαχωρισμό backend–GUI και καθαρή
αρχιτεκτονική.

------------------------------------------------------------
2. ΑΠΑΙΤΗΣΕΙΣ ΣΥΣΤΗΜΑΤΟΣ
------------------------------------------------------------

• Python 3.10 ή νεότερη  
• Δεν απαιτούνται εξωτερικές βιβλιοθήκες  
• Το GUI βασίζεται αποκλειστικά στο tkinter (ενσωματωμένο στην Python)

------------------------------------------------------------
3. ΟΔΗΓΙΕΣ ΕΓΚΑΤΑΣΤΑΣΗΣ
------------------------------------------------------------

1. Κατεβάστε το συμπιεσμένο αρχείο .zip του project.
2. Αποσυμπιέστε το σε έναν φάκελο της επιλογής σας.
3. Βεβαιωθείτε ότι η δομή των φακέλων είναι η εξής:

project/
│
├─ main.py
│
├─ backend/
│   ├─ models.py
│   ├─ manager.py
│   ├─ stats.py
│   ├─ recommendations.py
│   └─ storage_txt.py
│
├─ gui/
│   ├─ app.py
│   ├─ members_view.py
│   ├─ books_view.py
│   ├─ loans_view.py
│   ├─ stats_view.py
│   └─ recommend_view.py
│
└─ data/
    ├─ members.txt
    ├─ books.txt
    └─ loans.txt

4. Βεβαιωθείτε ότι τα αρχεία στο φάκελο data υπάρχουν (ακόμη και κενά).

------------------------------------------------------------
4. ΤΡΟΠΟΣ ΕΚΤΕΛΕΣΗΣ ΤΗΣ ΕΦΑΡΜΟΓΗΣ
------------------------------------------------------------

Ανοίξτε τερματικό μέσα στον φάκελο του project και εκτελέστε:

    python main.py

Το γραφικό περιβάλλον θα ανοίξει αυτόματα.

------------------------------------------------------------
5. ΠΕΡΙΓΡΑΦΗ ΛΕΙΤΟΥΡΓΙΩΝ
------------------------------------------------------------

5.1 Διαχείριση Μελών
• Προσθήκη νέου μέλους  
• Προβολή λίστας μελών  
• Ανανέωση λίστας  
• Αποθήκευση αλλαγών σε αρχείο members.txt  

5.2 Διαχείριση Βιβλίων
• Προσθήκη νέου βιβλίου  
• Προβολή λίστας βιβλίων  
• Ανανέωση λίστας  
• Αποθήκευση αλλαγών σε αρχείο books.txt  

5.3 Διαχείριση Δανεισμών
• Δημιουργία νέου δανεισμού  
• Επιστροφή βιβλίου  
• Καταγραφή rating (προαιρετικό)  
• Αποθήκευση αλλαγών σε loans.txt  

5.4 Στατιστικά
• Στατιστικά ανά μέλος  
• Στατιστικά ανά κατηγορία  
• Στατιστικά ανά συγγραφέα  
• Στατιστικά ανά ηλικία  
• Στατιστικά ανά φύλο  

5.5 Recommendation System
Το σύστημα προτείνει βιβλία σε κάθε μέλος με βάση:
• Προτιμήσεις κατηγοριών  
• Συχνότητα δανεισμών  
• Δημοτικότητα βιβλίων  
• Αποφυγή βιβλίων που έχει ήδη δανειστεί  

------------------------------------------------------------
6. ΑΡΧΕΙΑ .TXT
------------------------------------------------------------

Τα δεδομένα αποθηκεύονται σε μορφή pipe-separated (|).

6.1 members.txt
member_id|name|address|phone|email|age|profession|gender|active

6.2 books.txt
book_id|title|author|category|isbn|stock|total_loans

6.3 loans.txt
loan_id|member_id|book_id|loan_date|return_date|status|rating

------------------------------------------------------------
7. ΓΝΩΣΤΟΙ ΠΕΡΙΟΡΙΣΜΟΙ
------------------------------------------------------------

• Δεν υπάρχει έλεγχος για διπλότυπα ISBN.  
• Δεν υπάρχει undo/redo.  
• Το GUI δεν υποστηρίζει dark mode.  
• Τα στατιστικά εμφανίζονται σε text box και όχι σε γραφήματα.
